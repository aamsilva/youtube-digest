# 🎬 YouTube Intelligence Newsletter

Daily curated YouTube videos with AI-powered analysis and personal interest scoring.

**🌐 Live Version**: https://aamsilva.github.io/youtube-digest/

## Features

- 🤖 **AI-Powered Analysis**: Every video analyzed for topic categorization
- 🎯 **Personal Interest Scoring**: Priority rating based on your preferences (AI/Tech heavy)
- 🏷️ **Smart Categories**: AI/Tech, Finance, Crypto, Science, Geopolitics, etc.
- 📝 **Auto-Summaries**: 100-word AI-generated summaries
- 🔗 **Direct Links**: One-click YouTube access
- 🎨 **Beautified UI**: Dark theme with interactive filtering

## How It Works

1. **Import**: YouTube videos are automatically transcribed
2. **Analyze**: AI categorizes content and generates summaries
3. **Score**: Videos are ranked by personal interest level
4. **Publish**: Daily newsletter generated and hosted here
